# Your Energy Best - Website

Professional solar & HVAC solutions for San Diego homeowners.

## 🏠 Site Structure

- `index.html` - Home page
- `quote.html` - Quote request form (Zoho CRM integration)
- `build/` - Recruiting page
- `blog.html` - Blog posts
- `admin.html` - Content management panel (password: beacons)
- `content.json` - All site content (editable via admin)
- `style.css` - Global styling
- `cms.js` - Content rendering
- `img/` - Images and logo
