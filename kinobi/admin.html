This opens the presentation in a new tab with only your selected slides visible, then triggers the browser print dialog. Choose "Save as PDF" and set paper size to Landscape / Letter for best results.

Tip: In the print dialog → More settings → Layout: Landscape → Paper size: Letter → Margins: None → Enable "Background graphics"
